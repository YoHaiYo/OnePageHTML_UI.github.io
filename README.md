# 소개
html 한 페이지로 구성되어 복붙만하면 실행될 수 있게 작성된 초보자용 UI 모음입니다. 
공부하며 이건 ‘다시 써먹을 수 있겠다.’, ‘다시 복습할만하다’ 싶은 UI 코드들을 정리해 놓은 사이트입니다.

# 프로젝트 특징
- html 한 페이지로 구성. 코드를 복사, 붙여넣기해서 빠르게 확인 가능.
- 깃허브 파일과 링크연동.
